Requisitos

¿Qué necesito para empezar a utilizar Banca Online?

SISTEMAS OPERATIVOS
  • Windows: 7 o versiones superiores.
  • Mac: OS X Yosemite 10.10 o versiones posteriores.
  • Linux: versión de 64 bits de Ubuntu 14.04, Debian 8, openSUSE 13.3 o Fedora Linux 24 o versiones posteriores.
NAVEGADORES
  • Chrome 60.
  • Firefox 55.
  • IE 10.
  • Safari 5.
OTROS REQUISITOS
  • Procesador: Pentium 4 o superior.
  • Memoria RAM: 2GB.
  • Velocidad de navegación: 1MB.
  • Cookies y Java habilitado.
