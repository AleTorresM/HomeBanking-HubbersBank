Requirements

What do I need to start using Online Banking?

OPERATING SYSTEMS
  • Windows: 7 or higher versions.
  • Mac: OS X Yosemite 10.10 or later.
  • Linux: 64-bit version of Ubuntu 14.04, Debian 8, openSUSE 13.3, or Fedora Linux 24 or later.
BROWSER
  • Chrome 60.
  • Firefox 55.
  • IE 10.
  • Safari 5.
OTHER REQUIREMENTS
  • Processor: Pentium 4 or higher.
  • RAM memory: 2GB.
  • Browsing speed: 1MB.
  • Cookies and Java enabled.
